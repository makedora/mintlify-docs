---
title: File Info
api: GET https://api.yolocode.ai/api/e2b/{sandboxId}/files/info
description: Get file metadata from a sandbox
---

Returns file metadata from a sandbox.

<ParamField header="Authorization" type="string" required>Bearer token. `Bearer <github_token>`</ParamField>

<ParamField path="sandboxId" type="string" required>Sandbox ID.</ParamField>
<ParamField query="path" type="string" required>File path.</ParamField>

<Note>For Daytona sandboxes, use `/api/daytona/{sandboxId}/files/info`.</Note>
