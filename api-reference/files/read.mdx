---
title: Read File
api: GET https://api.yolocode.ai/api/e2b/{sandboxId}/files/read
description: Read file contents from a sandbox
---

Reads file contents from a sandbox.

<ParamField header="Authorization" type="string" required>Bearer token. `Bearer <github_token>`</ParamField>

<ParamField path="sandboxId" type="string" required>Sandbox ID.</ParamField>
<ParamField query="path" type="string" required>File path to read.</ParamField>

<Note>For Daytona sandboxes, use `/api/daytona/{sandboxId}/files/read`.</Note>
