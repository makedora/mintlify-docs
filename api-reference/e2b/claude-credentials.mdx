---
title: Set Claude Credentials
api: POST /api/e2b-sandboxes/{id}/claude-credentials
description: Configure Claude OAuth credentials in the sandbox
---

Writes Claude credentials to `.claude/.credentials.json` inside the sandbox and restarts the Claude WebSocket server.

<ParamField path="id" type="string" required>Sandbox ID.</ParamField>
<ParamField body="accessToken" type="string" required>Claude access token.</ParamField>
<ParamField body="refreshToken" type="string" required>Claude refresh token.</ParamField>
<ParamField body="expiresAt" type="string" required>Token expiry (ISO 8601).</ParamField>
<ParamField body="account" type="object">Account info with `id` and `name`.</ParamField>
<ParamField body="selectedModel" type="string">Preferred Claude model.</ParamField>
<ParamField body="subscriptionInfo" type="object">Subscription details.</ParamField>

### Response
Returns confirmation that credentials were set.
