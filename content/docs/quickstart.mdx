---
title: Quickstart
description: Get up and running with Hyperbrowser in under 5 minutes
---

## 1. Get an API key

Sign up at [app.hyperbrowser.ai](https://app.hyperbrowser.ai) and set your key:

```bash
export HYPERBROWSER_API_KEY=your-api-key-here
```

## 2. Install

```bash
npm install @hyperbrowser/sdk
```

## 3. Scrape a page

```typescript
import Hyperbrowser from "@hyperbrowser/sdk";

const client = new Hyperbrowser({
  apiKey: process.env.HYPERBROWSER_API_KEY,
});

const result = await client.scrape.startAndWait({
  url: "https://news.ycombinator.com",
});
console.log(result.data);
```

## 4. Extract structured data

```typescript
const result = await client.extract.startAndWait({
  urls: ["https://news.ycombinator.com"],
  schema: {
    type: "object",
    properties: {
      stories: {
        type: "array",
        items: {
          type: "object",
          properties: {
            title: { type: "string" },
            url: { type: "string" },
            points: { type: "number" },
          },
        },
      },
    },
  },
  prompt: "Extract the top 5 stories",
});
console.log(result.data);
```

## 5. Manual session (Playwright)

For full browser control, create a session and connect via CDP:

```typescript
import { chromium } from "playwright-core";

const session = await client.sessions.create();
try {
  const browser = await chromium.connectOverCDP(session.wsEndpoint);
  const page = browser.contexts()[0].pages()[0];
  await page.goto("https://news.ycombinator.com");
  console.log(await page.title());
} finally {
  await client.sessions.stop(session.id);
}
```

<Note>
**Pitfalls:**
- Package: `@hyperbrowser/sdk` (Node.js) / `hyperbrowser` (Python)
- Env var: `HYPERBROWSER_API_KEY`
- Sessions: always `sessions.stop()` in a `finally` block
- CDP endpoint: `session.wsEndpoint` (not `session.url`)
- Prefer `scrape.startAndWait()` over manual sessions for simple tasks
</Note>
