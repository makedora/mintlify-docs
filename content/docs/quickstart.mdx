## Setup

```bash
npm install @onkernel/sdk playwright-core
export KERNEL_API_KEY=your-api-key    # get one at dashboard.onkernel.com
```

## Complete working example

Copy this file and run it. It creates a cloud browser, navigates to Hacker News, extracts the top 5 story titles, and cleans up.

```typescript
import Kernel from "@onkernel/sdk";
import { chromium } from "playwright-core";

async function main() {
  const kernel = new Kernel();
  const kb = await kernel.browsers.create({ timeout_seconds: 300 });

  // CDP proxy may need a moment — retry connection
  let browser;
  for (let i = 0; i < 3; i++) {
    try { browser = await chromium.connectOverCDP(kb.cdp_ws_url); break; }
    catch (e) { if (i === 2) throw e; await new Promise(r => setTimeout(r, 2000)); }
  }

  const page = browser!.contexts()[0].pages()[0];
  await page.goto("https://news.ycombinator.com");

  // Extract top 5 story titles — use string literals, not arrow functions
  const titles = await page.evaluate(`
    Array.from(document.querySelectorAll('.titleline > a'))
      .slice(0, 5)
      .map(a => a.textContent)
  `);

  console.log("Top 5 HN stories:", titles);
  await browser!.close();
}

main();
```
