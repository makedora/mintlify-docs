[Notte](https://notte.cc) provides cloud browser infrastructure for AI agents — instant sessions with anti-detection, proxies, and captcha solving. Three main capabilities:

- **Browser sessions** — Playwright over CDP for full browser control
- **Web agents** — autonomous browsing that completes tasks without selectors
- **Scraping** — structured data extraction with Pydantic models

## Prerequisites

- Notte account at [console.notte.cc](https://console.notte.cc)
- API key from the console dashboard

<Warning>
Set your API key as `NOTTE_API_KEY` environment variable before running any code. The SDK reads it automatically.
</Warning>

```bash
export NOTTE_API_KEY=your-api-key-here
```

## Installation

```bash
# Python (requires >= 3.11)
pip install notte-sdk

# JavaScript
npm install notte-sdk
```

<Note>
For browser sessions, also install Playwright: `pip install playwright`
</Note>

## Browser Session (Playwright over CDP)

The most common integration pattern — connect Playwright to a cloud browser via CDP.

```python
from playwright.sync_api import sync_playwright
from notte_sdk import NotteClient

client = NotteClient()

with client.Session() as session:
    cdp_url = session.cdp_url()

    with sync_playwright() as p:
        browser = p.chromium.connect_over_cdp(cdp_url)
        page = browser.contexts[0].pages[0]
        page.goto("https://news.ycombinator.com")
        title = page.title()
        print(f"Page title: {title}")
        page.screenshot(path="screenshot.png")
```

<Tip>
Use `open_viewer=True` in `client.Session()` to get a live view URL for debugging: `client.Session(open_viewer=True)`
</Tip>

## Web Agent

For autonomous task completion without writing selectors:

```python
from notte_sdk import NotteClient

client = NotteClient()

with client.Session() as session:
    agent = client.Agent(session=session, max_steps=5)
    response = agent.run(
        task="Find the top 5 stories on Hacker News",
        url="https://news.ycombinator.com"
    )
    print(response)
```

## Scraping with Pydantic

For structured data extraction:

```python
from pydantic import BaseModel
from notte_sdk import NotteClient

class Story(BaseModel):
    title: str
    url: str
    points: int

class Feed(BaseModel):
    stories: list[Story]

client = NotteClient()
result = client.scrape(
    url="https://news.ycombinator.com",
    response_format=Feed,
    instructions="Extract the top 5 posts"
)

for i, story in enumerate(result.data.stories, 1):
    print(f"{i}. {story.points}pts - {story.title}")
```

## Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `NOTTE_API_KEY is NOT SET` | Missing environment variable | Run `export NOTTE_API_KEY=your-key`. Get key from [console.notte.cc](https://console.notte.cc) |
| `'Page.enable' wasn't found` (CDP -32601) | CDP version mismatch with remote browser | Use `connect_over_cdp()` not `connect()`. Ensure you're using `p.chromium` not `p.firefox` |
| Session timeout / no response | Session expired or network issue | Sessions auto-close after inactivity. Create a new session and retry |
| `playwright._impl._errors.Error: Browser closed` | Session ended before script finished | Wrap code in `with client.Session()` context manager to ensure proper lifecycle |
