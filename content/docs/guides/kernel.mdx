title: Kernel Browser API

Kernel provides cloud browsers via a simple API — create a browser, get a CDP WebSocket URL, connect with Playwright or Puppeteer. Each browser runs in its own VM with up to 72-hour sessions. Formerly known as OnKernel.

## Prerequisites

- Kernel account at [dashboard.onkernel.com](https://dashboard.onkernel.com/sign-up)
- API key from the dashboard
- Kernel CLI (optional, for deploying apps)

<Warning>
Set your API key as `KERNEL_API_KEY` environment variable. The SDK reads it automatically.
</Warning>

```bash
export KERNEL_API_KEY=your-api-key-here
```

## Installation

```bash
# TypeScript SDK
npm install @onkernel/sdk

# Python SDK
uv pip install kernel

# CLI (optional — for deploying apps)
brew install onkernel/tap/kernel
```

<Note>
The npm package is `@onkernel/sdk` (not `kernel`). The Python package is `kernel`.
</Note>

## Quick Start: Browser + Playwright

<Tabs>
  <Tab title="TypeScript">
    ```typescript
    import Kernel from "@onkernel/sdk";
    import { chromium } from "playwright";

    const kernel = new Kernel();
    const kernelBrowser = await kernel.browsers.create();
    const browser = await chromium.connectOverCDP(kernelBrowser.cdp_ws_url);

    const page = browser.contexts()[0].pages()[0];
    await page.goto("https://news.ycombinator.com");
    const title = await page.title();
    console.log(`Page title: ${title}`);
    await page.screenshot({ path: "screenshot.png" });

    await browser.close();
    ```
  </Tab>
  <Tab title="Python">
    ```python
    from kernel import Kernel
    from playwright.async_api import async_playwright

    kernel = Kernel()
    kernel_browser = kernel.browsers.create()

    async with async_playwright() as playwright:
        browser = await playwright.chromium.connect_over_cdp(kernel_browser.cdp_ws_url)
        page = browser.contexts[0].pages[0]
        await page.goto("https://news.ycombinator.com")
        title = await page.title()
        print(f"Page title: {title}")
        await page.screenshot(path="screenshot.png")
        await browser.close()
    ```
  </Tab>
</Tabs>

<Tip>
The key return value is `cdp_ws_url` — a Chrome DevTools Protocol WebSocket URL. Pass it to `connectOverCDP()` (TypeScript) or `connect_over_cdp()` (Python).
</Tip>

## Deploy as an App (CLI)

For serverless deployment:

```bash
# Create a new app
kernel create

# Deploy
kernel deploy index.ts

# Invoke
kernel invoke my-app my-function --payload '{"url": "https://example.com"}'
```

<Note>
Use `kernel login` to authenticate the CLI before deploying.
</Note>

## Troubleshooting

| Error | Cause | Fix |
|-------|-------|-----|
| `KERNEL_API_KEY` not found | Missing environment variable | Run `export KERNEL_API_KEY=your-key`. Get key from [dashboard.onkernel.com](https://dashboard.onkernel.com) |
| 404 on docs pages | Many `/docs/*` paths don't exist yet | Use `https://www.kernel.sh/docs/llms.txt` for the full API reference index |
| `cdp_ws_url` is undefined | Browser creation failed silently | Check your API key is valid and you haven't hit concurrent session limits |
| Browser disconnects unexpectedly | Session timed out (default varies) | Set a longer timeout: `kernel.browsers.create({ timeout: 3600 })` for up to 72h sessions |
| `@onkernel/sdk` not found | Wrong package name | TypeScript: `@onkernel/sdk`. Python: `kernel`. Do not use `kernel-sdk` or `onkernel` |
| `kernel` CLI command not found | CLI not installed | Run `brew install onkernel/tap/kernel` |
