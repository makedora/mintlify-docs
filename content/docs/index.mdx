Kernel gives you cloud browsers through a simple API. Create a browser, get a CDP WebSocket URL, connect with Playwright or Puppeteer. Each browser runs in its own VM with up to 72-hour sessions. Formerly known as OnKernel.

## Quick example

```typescript
import Kernel from "@onkernel/sdk";
import { chromium } from "playwright";

const kernel = new Kernel();
const browser = await kernel.browsers.create();
const pw = await chromium.connectOverCDP(browser.cdp_ws_url);
const page = pw.contexts()[0].pages()[0];
await page.goto("https://news.ycombinator.com");
console.log(await page.title());
```
