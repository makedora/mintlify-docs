Hyperbrowser gives you cloud browsers for AI agents and automation. Scrape websites, extract structured data, run autonomous browser agents — all with built-in CAPTCHA solving, anti-detection, and proxies.

## Quick start

```bash
npm install @hyperbrowser/sdk
export HYPERBROWSER_API_KEY=your-key    # get one at app.hyperbrowser.ai
```

```typescript
import Hyperbrowser from "@hyperbrowser/sdk";

const client = new Hyperbrowser({
  apiKey: process.env.HYPERBROWSER_API_KEY,
});

const result = await client.scrape.startAndWait({
  url: "https://news.ycombinator.com",
});
console.log(result.data);
```
