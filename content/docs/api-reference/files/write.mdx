title: Write File
description: Write content to a file in a sandbox

Writes content to a file in a sandbox.

<ParamField header="Authorization" type="string" required>Bearer token. `Bearer <github_token>`</ParamField>

<ParamField path="sandboxId" type="string" required>Sandbox ID.</ParamField>
<ParamField body="path" type="string" required>File path to write.</ParamField>
<ParamField body="content" type="string" required>File content.</ParamField>

<Note>For Daytona sandboxes, use `/api/daytona/{sandboxId}/files/write`.</Note>
