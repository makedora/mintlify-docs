---
title: Transcribe (Streaming)
description: Stream audio transcription via NVIDIA Parakeet
---

Accepts audio as base64 JSON or multipart form data.

<ParamField header="Authorization" type="string" required>Bearer token. `Bearer <github_token>`</ParamField>

<ParamField body="audio" type="string" required>Base64-encoded audio data (JSON mode) or audio file (multipart).</ParamField>
<ParamField body="languageCode" type="string" default="en-US">BCP-47 language code.</ParamField>

Supported formats: WAV, FLAC, PCM. Max duration: 300 seconds.
